# Постановка задачи

Цель: разработать WPF‑приложение для учета и контроля библиотечного фонда.

Функции:

* учет книг и экземпляров
* учет читателей
* выдача и возврат книг
* контроль просрочек
* поиск и фильтрация
* бронирование книг
* базовая отчетность

Ограничения:

* архитектура MVVM
* Entity Framework Core
* локальная БД (MS SQL Server)

Критерий успеха: корректный учет движения книг и наглядный контроль состояния фонда.
