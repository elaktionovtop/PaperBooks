# Интерфейс приложения

## Окна приложения

### Окно авторизации

* стандартное диалоговое окно
* ввод логина и пароля
* определение роли пользователя (библиотекарь / администратор)
* вход без логина и пароля — режим читателя
* при входе читателя сразу открывается окно поиска

### Главное окно

Содержит основные рабочие области и меню.

#### Рабочие области

**Список читателей**

* список зарегистрированных читателей
* при выборе читателя отображаются книги, находящиеся у него на руках

**Журнал выдачи книг**

* отображаются только записи с невозвращёнными книгами
* используется для контроля текущих выдач и просрочек

**Список книг**

* список всех книг фонда
* для выбранной книги:

  * список читателей, забронировавших книгу
  * общее количество экземпляров
  * количество свободных экземпляров

#### Меню

* команды библиотекаря:

  * открытие окон редактирования данных
  * открытие окна поиска
  * другие служебные команды
* команды администратора:

  * управление пользователями
  * доступны только при входе администратора

### Окна редактирования данных

* список книг
* список экземпляров
* список авторов

Примечание:

* операции выдачи и возврата выполняются в главном окне

### Окно поиска

* поиск книг:

  * по названию
  * по автору
* бронирование книги
* отмена бронирования

## Общие требования к интерфейсу

* WPF + MVVM
* привязки данных (DataBinding)
* команды (ICommand)
* отсутствие бизнес‑логики в code‑behind
